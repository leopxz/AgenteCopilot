# 🌍 Leonardo Viagens – AI Travel Visa Specialist

An intelligent travel documentation assistant built using Microsoft Copilot Studio and Power Platform.

## 📌 Project Overview

Leonardo Viagens AI Agent is a virtual travel documentation specialist designed to assist Brazilian passport holders with international visa requirements and travel regulations.

The agent provides structured, reliable, and professional guidance on:

- Visa requirements per country
- Types of visas (tourism, business, study, work, transit)
- Required travel documentation
- Maximum stay duration
- Entry requirements and immigration rules
- Application process guidance
- Official government and consulate references

This project demonstrates practical implementation of Generative AI within Microsoft Copilot Studio.

---

## 🧠 Technology Stack

- Microsoft Copilot Studio
- Microsoft Power Platform
- Dataverse
- Generative AI (LLM-based natural language understanding)
- Structured Knowledge Base (Travel Visa Data)

---

## 🏗 Architecture

User → Copilot Agent →  
- System Instructions (Role-based Prompt Engineering)  
- Knowledge Base (Travel Documentation Data)  
- Generative AI Engine  
- Structured Response Framework  

---

## 🎯 Agent Design Strategy

The agent was built with:

- Role-based system prompt locking
- Structured response formatting
- Clarification protocol (asks for missing travel details)
- Anti-hallucination safeguards
- Professional tone aligned with a travel consultancy brand

---

## 📋 Response Structure

Each answer follows this structure:

1. Visa Requirement  
2. Type of Visa  
3. Maximum Stay  
4. Required Documents  
5. Application Process  
6. Important Notes  

---

## 🔐 Compliance & Safety

The agent:
- Avoids speculative legal guarantees
- Recommends official government sources when regulations may change
- Does not fabricate immigration policies
- Maintains professional consultancy tone
